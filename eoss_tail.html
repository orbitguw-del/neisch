Step 3 - Scheme

Dual GST: EOSS side = actual (file column → else slab by date & selling price: ≤22 Sep 2025 5% to Rs1000 else 12%; from 22 Sep 2025 5% to Rs2500 else 18% → else this default). Normal side uses the slab the MRP falls in.
Used as the dealer when the file has no store column or it is blank.
If goods were billed to the dealer before 22 Sep, pick Pre-22 Sep so GST-on-MRP uses 12%, not the sale-day 18%.
