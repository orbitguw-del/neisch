CONTRACTOR · PART 1

Contractor — How To

Setting up the company, sites and team.

1 Register your company
Open the app or storeyinfra.com.
On the sign-in screen tap Register your company.
Enter your name, company name, email and a password.
Tap Create account — you're in as the Contractor.
2 Add a construction site
Menu → Sites → tap Add Site.
Enter site name, location, start date and budget.
Set status (Planning / Active) → Save.
3 Add a worker
Menu → Workers → tap Add Worker.
Enter name, trade, daily wage and phone.
Choose Direct hire or Vendor (enter vendor name if vendor).
Assign the worker to a site.
ID proof: pick type, enter only the last 4 digits → Add Worker.
Privacy: never enter the full Aadhaar/PAN — last 4 digits only.
4 Invite a team member
Menu → Team → tap Invite.
Enter their email, pick the role (Site Manager / Supervisor / Store Keeper).
Assign the site(s) they will handle.
Send — they get an invite code to join with.
5 Set a budget line
Menu → Reports → Budget vs Actual tab.
Pick the site and month → tap Add budget line.
Choose a material, enter planned quantity and cost → Save.
Why: budget lines power the variance report — planned vs actually spent.

Day-to-day records, expenses and reports.

Menu → Attendance. Pick the site and date.
Tap each worker to cycle: Present → Absent → Half-day → Paid-leave.

Pick site, date, category (Fuel, Transport, Rental…), amount.

Review amount, category and note.
Tap Approve or Reject.

SITE MANAGER

Site Manager — How To

Running your assigned sites — workers, materials, expense approvals.

1 Add a worker to your site
Menu → Workers → Add Worker.
Fill name, trade, daily wage; assign to your site.
ID proof — last 4 digits only → Add Worker.
2 Mark attendance
Menu → Attendance. Pick site and date.
Tap each worker to set status → Save.
3 Write a daily log
Menu → Daily Logs → New log.
Enter date, workers present, weather, work done and issues → Save.
4 Record receipts & transfers
Receipts → record incoming materials with cost.
Transfers → move stock to/from another site.
Equipment → register machines and maintenance.
5 Add an expense
Menu → Expenses → Add Expense.
Enter site, date, category, amount, note → Add Expense.
6 Approve a Supervisor's expense
Menu → Expenses → filter Status = Pending.
Check the entry, then tap Approve or Reject.
Do this daily so reports stay accurate.

SITE SUPERVISOR

Site Supervisor — How To

Your on-ground daily tasks.

1 Mark daily attendance (every morning)
Menu → Attendance. Today's date is selected.
Tap each worker to cycle through P / A / ½ / PL.
Tap Mark all present first if most are in, then fix the rest.
Tap Save before leaving the screen.
2 Write the daily log (every evening)
Menu → Daily Logs → New log.
Enter how many workers were present.
Write what work was done, the weather, and any issues or delays.
Tap Save.
3 Raise a site expense (when cash is spent)
Menu → Expenses → Add Expense.
Pick the site and date.
Choose a category — Fuel, Transport, Food, Repairs, etc.
Enter the amount, who paid, and a short note.
Tap Add Expense — it goes to your Site Manager as Pending.
Do it immediately: log the expense when the money is spent, not later — small spends get forgotten.
4 Add a worker
Menu → Workers → Add Worker.
Enter name, trade, daily wage; assign to the site → Add Worker.

STORE KEEPER

Store Keeper — How To

Keeping materials and equipment accurate.

1 Add a material to inventory
Menu → Inventory → Add Material.
Enter name, unit (bag, ton, cft…), opening stock and reorder level.
Set the unit cost → Save.
2 Record a material receipt (GRN)
Menu → Receipts → Record receipt.
Pick the site and material.
Enter quantity received, supplier name and unit cost.
Tap Save — stock rises automatically.
Same-day entry: record receipts the day goods arrive so stock always matches reality.
3 Transfer materials to another site
Menu → Transfers → New transfer.
Choose From-site, To-site, the material and quantity.
Tap Save — both sites' stock update together.
4 Add equipment / an asset
Menu → Equipment → Add Equipment.
Enter name, type, and the site it's at.
Record assignment and any maintenance notes → Save.
5 Check stock levels
Menu → Inventory — see live quantity for every material.
Items at or below the reorder level need restocking — raise it with your Site Manager.
Tap a material to see its full receipt/transfer history (ledger).

QUICK REFERENCE

Daily Rhythm

If everyone does their bit each day, the reports take care of themselves.

Site Supervisor — every day
Morning: mark attendance for all workers.
Through the day: raise expenses as cash is spent.
Evening: write the daily log.
Store Keeper — as it happens
Record every material receipt the day it arrives.
Log transfers immediately so both sites stay correct.
Watch reorder levels; flag low stock early.
Site Manager — every day
Clear all Pending expenses — Approve or Reject.
Check attendance was marked for each site.
Contractor — weekly
Open Reports → Site Report for each site — review total spend.
Check Budget vs Actual — catch overspend early.
Print or PDF the reports you need to keep.
